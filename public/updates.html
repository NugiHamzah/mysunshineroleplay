📋 Changelog v1.2.1 — Harvest

🐞 Fixed

  • Personal Vehicle Despawn — Memperbaiki bug yang menyebabkan kendaraan pribadi terkadang hilang atau tidak muncul pada menu /spawn maupun /mv.
  • Mechanic Skill Reset — Memperbaiki bug dimana level Mechanic kembali menjadi Level 1 setelah mencapai Level 5 ketika melakukan modifikasi kendaraan melalui /vm.
  • /spec Crash — Memperbaiki bug yang dapat menyebabkan server crash saat menggunakan command /spec.
  • /unban Command — Memperbaiki command /unban yang sebelumnya tidak dapat digunakan.
  • Server Logo HUD — Memperbaiki bug dimana logo server kembali muncul setelah beberapa kali login meskipun telah disembunyikan melalui /hudmenu.
  • Modshop Reset — Memperbaiki bug yang menyebabkan seluruh modifikasi kendaraan hilang setelah melakukan perubahan di Modshop.
  • 3D Ammo Text — Memperbaiki bug visual dimana tulisan 3D Ammo masih terlihat walaupun telah dinonaktifkan.
  • Character Story — Memperbaiki status Character Story yang tetap menampilkan None meskipun CS telah berhasil diaktifkan.
  • Mobile Phone Store — Memperbaiki error "Invalid Item" saat membeli Handphone di toko.
  • Bus Driver Salary — Memperbaiki bug dimana gaji dari rute Bus A maupun Bus B tidak masuk ke paycheck pemain.
  • Job Vehicle Engine — Memperbaiki bug yang menyebabkan kendaraan job Courier, Sweeper, dan Trashmaster tidak dapat dinyalakan.
  • Trashmaster Checkpoint — Memperbaiki checkpoint GPS yang tidak muncul setelah menggunakan command /loadtrash.
  • /stuck Command — Memperbaiki pilihan pada command /stuck yang sebelumnya tidak menjalankan fungsi sesuai pilihannya.
  • Faction Vehicle Detection — Memperbaiki bug yang menyebabkan kendaraan pribadi dianggap sebagai kendaraan faction sehingga muncul pesan:

    "You're Not Members of This Faction."

🆕 Added

  • New Dealership Vehicles — Menambahkan kendaraan Sanchez dan PCJ-600 ke dealership/server.
  • Custom Ammo Withdrawal — Menambahkan sistem baru untuk mengambil ammunition dari bagasi kendaraan, dimana pemain kini dapat menentukan jumlah ammo yang ingin diambil.

📝 Catatan

Harvest v1.2.1 berfokus pada peningkatan stabilitas server, perbaikan sistem kendaraan, command administrator, sidejob, HUD, Character Story, hingga optimasi berbagai bug yang dilaporkan oleh pemain. Selain itu, update ini juga menghadirkan kendaraan baru di Dealership serta sistem pengambilan ammunition yang lebih fleksibel dari bagasi kendaraan.

Apabila masih menemukan bug atau kendala lainnya, silakan laporkan melalui halaman Report agar dapat segera kami tindak lanjuti.

📋 Changelog v1.1.11 — Stability Update

🐞 Fixed

  • Kendaraan Warga Tidak Bisa Dinaiki — Memperbaiki bug yang menyebabkan sebagian kendaraan pribadi tidak dapat dinaiki akibat sistem salah mengenali nilai kosong/netral. Validasi telah dioptimalkan agar masalah ini tidak kembali terjadi.
  • Kendaraan Masuk Asuransi Saat Logout — Memperbaiki bug yang menyebabkan kendaraan pemain otomatis masuk ke sistem asuransi ketika pemain logout. Penyebabnya adalah kendaraan tidak tercatat dengan benar saat player keluar sehingga dianggap hancur oleh server. Sistem telah diperbaiki dan masih akan terus dipantau pada update berikutnya.
  • Duplikasi Kendaraan — Mengoptimalkan sistem validasi kendaraan untuk menutup celah yang dapat menyebabkan kendaraan terduplikasi (duplicate vehicle). Data kendaraan lama yang terdampak akan dibersihkan secara manual melalui database apabila masih ditemukan.
  • Job Sweeper & Courier — Memperbaiki bug yang membuat pemain tidak dapat menyalakan kendaraan job Sweeper maupun Courier karena proses validasi kendaraan job sebelumnya terlewati dan langsung masuk ke pengecekan kendaraan pribadi.
  • Command /setcs — Memperbaiki bug dimana status Character Story masih menampilkan None setelah admin menggunakan perintah /setcs. Kini status akan langsung aktif secara otomatis.
  • Command /unspec — Memperbaiki bug yang dapat memicu crash akibat server membaca indeks negatif serta beberapa callback yang berpotensi menyebabkan benturan memori. Perbaikan telah diterapkan dan akan terus dipantau.
  • Command /spawn — Memperbaiki bug dimana kendaraan tidak dapat dimunculkan dengan benar dan justru dapat melakukan spawn ulang pada kendaraan yang sudah ada sehingga berpotensi menyebabkan duplikasi kendaraan. Validasi kendaraan yang sedang digunakan juga telah diperketat.
  • False Detection Faction Vehicle — Memperbaiki bug yang menyebabkan sebagian kendaraan pribadi terdeteksi sebagai kendaraan faction sehingga tidak dapat digunakan oleh pemiliknya.

    Demonstrasi bug sebelum dilakukan perbaikan.

📝 Catatan

"Update v1.1.11 berfokus pada peningkatan stabilitas sistem kendaraan dan server. Berbagai bug kritis seperti kendaraan tidak bisa dinaiki, kendaraan masuk asuransi saat logout, bug duplikasi kendaraan, validasi kendaraan job, command /setcs, /unspec, /spawn, hingga false detection kendaraan faction telah diperbaiki dan dioptimalkan. Beberapa sistem masih akan terus dipantau untuk memastikan kestabilan server ke depannya."

Apabila masih menemukan kendala, silakan laporkan melalui halaman Report agar dapat segera kami tindak lanjuti.

📋 Changelog v1.1.10 — Harvest

🆕 Added

  • Ikon Rumah di World Map — Rumah kini muncul pada World Map (TAB). Ikon Hijau menandakan rumah masih dijual, sedangkan ikon Merah menandakan rumah telah memiliki pemilik.
  • 3 Mapping Baru — Menambahkan 3 mapping baru hasil Owner Request untuk mempercantik dan meningkatkan pengalaman bermain di dalam kota.

🔄 Changed

  • Pasang Modifikasi Kendaraan — Proses pemasangan modifikasi di Mechanic kini hanya membutuhkan 10 detik (sebelumnya sekitar 1 menit 40 detik), sehingga proses modifikasi kendaraan menjadi jauh lebih cepat.

🐞 Fixed

  • Gaji Sidejob Milkman — Gaji Milkman kini menjadi $250 sesuai informasi yang ditampilkan pada dialog pekerjaan.
  • Kendaraan Rollback — Kendaraan pribadi tidak lagi mengalami rollback atau kembali ke posisi lama saat pemain keluar dari kendaraan maupun setelah server melakukan restart.
  • Kendaraan Dinas LSPD — Anggota SAPD kini dapat menggunakan kendaraan dinas LSPD tanpa mengalami kick otomatis.
  • Interior Rumah — Karakter tidak lagi jatuh menembus lantai saat memasuki interior rumah.
  • Reset Rumah — Setelah rumah di-reset oleh admin, nama pemilik lama kini langsung dihapus dan label rumah otomatis berubah menjadi "Dijual".
  • Rumah Double ID — Memperbaiki bug yang menyebabkan rumah yang telah dihapus admin masih muncul kembali atau memiliki ID ganda.
  • Character Story — Status Character Story pada perintah /stats kini tersimpan secara permanen dan tidak lagi hilang setelah disconnect maupun restart server.

📝 Catatan

"v1.1.10 — Harvest menghadirkan ikon rumah di World Map (TAB), 3 Mapping Baru hasil Owner Request, proses pemasangan modifikasi kendaraan yang kini hanya memerlukan 10 detik, serta berbagai perbaikan penting seperti gaji Milkman, kendaraan rollback, kendaraan dinas LSPD, interior rumah, reset rumah, Double ID rumah, dan penyimpanan permanen Character Story."

📋 Changelog v1.1.8 — Fortified

🆕 Added

  • Flare Multi-Warna /flare — Perintah /flare kini memiliki pilihan 3 warna: Merah (backup umum), Biru (prioritas), dan Hijau (aman). Flare juga sekarang terlihat dan terkirim ke seluruh faction pemerintah (SAPD, SAGS, SAMD, dan SANEWS).
  • Sharekey Kendaraan /sharekey — Pemilik kendaraan pribadi kini dapat membagikan akses kunci kepada pemain lain menggunakan /sharekey [playerid]. Pemegang kunci dapat membuka/mengunci kendaraan serta menyalakan mesin. Kendaraan yang sedang terkunci kini tidak dapat dinaiki maupun dinyalakan oleh pemain lain.
  • Damage Log — Setiap tembakan senjata api kini mencatat identitas penembak. Informasi tersebut dapat dilihat melalui /damagelog (kolom Shooter) maupun oleh tim medis melalui perintah /surgery.

🛠️ Fixed

  • Radio Faction SAMD & SANEWS — Anggota SAMD dan SANEWS kini kembali menerima seluruh pesan radio faction (/r, /or, /od, dan /d) dengan normal, termasuk saat sedang off-duty.
  • Pesan Radio Mati — Saat radio dalam keadaan mati, penggunaan /r kini akan memberikan petunjuk untuk menyalakan radio kembali dalam Bahasa Indonesia.
  • Departemen /d — Perintah /d kini dapat digunakan lintas seluruh faction pemerintah (SAPD, SAGS, SAMD, dan SANEWS), bukan hanya antar anggota faction yang sama.

📝 Catatan

"v1.1.8 — Radio faction SAMD & SANEWS telah diperbaiki, komunikasi /d kini lintas seluruh faction pemerintah, sistem flare mendapatkan 3 pilihan warna, fitur Sharekey kendaraan ditambahkan untuk meningkatkan keamanan kendaraan pribadi, serta Damage Log kini mencatat nama penembak secara lengkap."

📋 Changelog v1.1.7 — Stabilitas

🆕 Added

  • Pertolongan Pertama /salve — Tim EMS sekarang bisa memberikan pertolongan pertama kepada warga yang terluka tanpa membutuhkan medkit.

🛠️ Fixed

  • Dealer Lowrider — Kendaraan lowrider (Blade, Broadway, Hermes, Savanna, dan lainnya) sekarang sudah dapat dibeli melalui dealer lowrider.
  • Furniture Rumah Hilang — Furniture dan dekorasi rumah yang dipasang melalui /dyoh kini tidak akan hilang lagi setelah server restart.
  • Mancing di Perairan — Pemain kini dapat memancing di laut selama berada dekat kapal dan berada di area mancing, tidak hanya di area karnaval.
  • Skill Trucker — Skill trucker sekarang dapat meningkat hingga level 5 dengan normal. Sebelumnya hanya berhenti di level 2.
  • Locker Faction Bank — Sistem deposit uang pada locker faction (SAPD, SAGS, SAMD, dan SANEWS) kini kembali berfungsi dengan normal.
  • Skin Locker SANEWS — Karakter pria di faction SANEWS kini memiliki pilihan skin yang sesuai di locker.

📝 Catatan

"v1.1.7 — update stabilitas: perbaikan dealer lowrider, furniture rumah, mancing, skill trucker, locker faction, dan skin SANEWS."
